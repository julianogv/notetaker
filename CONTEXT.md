# Notetaker

CLI tool that records meetings, transcribes locally with Whisper, and generates a
structured Summary through an LLM Provider (external CLI). Audio never leaves
the machine; only the text transcription is sent to the LLM Provider.

## Language

**Meeting**:
A recorded session, from `start` to `stop`. Corresponds to a folder on disk.
_Avoid_: session, call, recording

**Track**:
An audio stream recorded separately. Online: `mic` (your voice) and `system`
(voice of participants, from the monitor output). In-person: `mic` only.
_Avoid_: channel, stream, band

**Mode**:
How audio is captured. `online` = mic + system; `in-person` = mic only;
`listener` = system only (you as a listener in an online meeting).
_Avoid_: meeting type

**Diarization**:
Attribute speech to who spoke. `level1` distinguishes you (Track mic) from
participants (Track system) without ML. `level2` uses ML to separate each speaker.
_Avoid_: speaker separation, speaker split

**Summary**:
Output generated by the LLM Provider from the transcription, in structured Markdown.
Persisted as `summary.md`.
_Avoid_: minutes, report, synthesis

**LLM Provider**:
External CLI command (defined in config) that receives the transcription via stdin
and returns the Summary in JSON. Notetaker does not talk to LLM APIs directly.
_Avoid_: model, AI, backend

**Meeting Language**:
Language spoken in the meeting (`auto`, `pt`, `es`, `en`). Determines the language of the
transcription and, by default, the language of the Summary.
_Avoid_: locale
